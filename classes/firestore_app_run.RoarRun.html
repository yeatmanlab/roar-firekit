Class RoarRun

Class representing a ROAR run.

A run is a globally unique collection of successive trials that constitute one user "running" through a single assessment one time.

writeInteractions

  • writeInteractions(assessmentStage: string, trialDocRef: DocumentReference<DocumentData>): Promise<void>
    • Defined in src/firestore/app/run.ts:268

    Writes interaction data for the current trial and updates run-level counters.

    This method saves the interaction data collected during the current trial to Firestore. It performs two writes:

    1. Updates the trial document with detailed interaction data (e.g., timestamps).
    2. Increments counters on the run document to track how many times each interaction type occurred (e.g., blur, focus) during a specific assessment stage (e.g., practice, test).

    After writing, the interaction data is reset for the next trial.

    Parameters

    • assessmentStage: string

      The current stage of the assessment (e.g., "practice", "test")

    • trialDocRef: DocumentReference<DocumentData>

      Reference to the Firestore document representing the current trial

    Returns Promise<void>
